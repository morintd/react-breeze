# Getting Started

react-breeze is a component library working with 6 variants: **primary**, **secondary**, **success**, **info**, **warning**, **danger**.

It allows for quick prototyping, using a set of configurable colors with a given intention.

import { Title } from 'react-breeze';

<Title as="h2">Installation</Title>

**yarn**

```shell
yarn add react-breeze
```

**npm**

```shell
npm add --save react-breeze
```

<Title as="h2">Configuration</Title>

Before using react-breeze, you need to [setup Tailwind CSS](https://tailwindcss.com/docs/installation).

Next, we need to configure colors and their variants, as well as allowing the variant _active_ for background color:
