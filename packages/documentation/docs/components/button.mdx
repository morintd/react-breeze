# Button

A Button suggest a user action.

import { Button, Title } from 'react-breeze';

<Title as="h3">Color</Title>

<Button className="mb-2 mr-1">Button (Primary)</Button>

</Button>
<Button className="mb-2 mr-1" color="success">

</Button>

</Button>
<Button className="mb-2 mr-1" color="warning">

</Button>
<Button className="mb-2 mr-1" color="danger">
  Button (Danger)
</Button>

**Properties**

Button takes the same properties as a `button`, but can also be given:

| Name     | Type                                                                     | Default   | Required | Description                                     |
| -------- | ------------------------------------------------------------------------ | --------- | -------- | ----------------------------------------------- |
| color    | `primary` \| `secondary` \| `success` \| `info` \| `warning` \| `danger` | `primary` |          | Set color                                       |
| outline  | `boolean`                                                                | `false`   |          | Alternative appearance                          |
| loading  | `boolean`                                                                | `false`   |          | Show a loader instead of `children`             |
| disabled | `boolean`                                                                | `false`   |          | Change appearance and doesn't trigger `onClick` |
