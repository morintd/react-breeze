# Button

A Button suggest a user action.

import { Button, Title } from 'react-breeze';

<Title as="h3">Color</Title>

<Button className="mb-2 mr-1" color="blue">
  Button (Blue)
</Button>
<Button className="mb-2 mr-1" color="red">
  Button (Red)
</Button>

```jsx
import { Button } from 'react-breeze';

<Button color="blue">Button (Blue)</Button>
<Button color="red">Button (Red)</Button>
```

<Title as="h3">Outline</Title>

Buttons can have a different appearance if they're outlined.

<Button outline className="mb-2 mr-1" color="blue">
  Button (Blue)
</Button>
<Button outline className="mb-2 mr-1" color="red">
  Button (Red)
</Button>

```jsx
import { Button } from 'react-breeze';

<Button outline color="blue">Button (Blue)</Button>
<Button outline color="red">Button (Red)</Button>
```

<Title as="h3">Loading</Title>

Buttons can display a loader, useful when working with async actions (default or outline).

<div>
  <Button loading className="mb-2 mr-1" color="blue">
    Button (Blue)
  </Button>
  <Button loading className="mb-2 mr-1" color="red">
    Button (Red)
  </Button>
</div>
<div>
  <Button outline loading className="mb-2 mr-1" color="blue">
    Button (Blue)
  </Button>
  <Button outline loading className="mb-2 mr-1" color="red">
    Button (Red)
  </Button>
</div>

```jsx
import { Button } from 'react-breeze';

<Button loading color="blue">Button (Blue)</Button>
<Button loading color="red">Button (Red)</Button>

<Button outline loading color="blue">Button (Blue)</Button>
<Button outline loading color="red">Button (Red)</Button>
```

<Title as="h3">Disabled</Title>

Buttons can have a disabled state, used to indicate they cannot be clicked (default or outline).

<div>
  <Button disabled className="mb-2 mr-1" color="blue">
    Button (Blue)
  </Button>
  <Button disabled className="mb-2 mr-1" color="red">
    Button (Red)
  </Button>
</div>
<div>
  <Button disabled outline className="mb-2 mr-1" color="blue">
    Button (Blue)
  </Button>
  <Button disabled outline className="mb-2 mr-1" color="red">
    Button (Red)
  </Button>
</div>

```jsx
import { Button } from 'react-breeze';

<Button disabled color="blue">Button (Blue)</Button>
<Button disabled color="red">Button (Red)</Button>

<Button outline disabled color="blue">Button (Blue)</Button>
<Button outline disabled color="red">Button (Red)</Button>
```

**Properties**

Button takes the same properties as a `button`, but can also be given:

| Name     | Type      | Default     | Required     | Description                                     |
| -------- | --------- | ----------- | ------------ | ----------------------------------------------- |
| color    | `string`  | `undefined` | **required** | Set color                                       |
| outline  | `boolean` | `false`     |              | Alternative appearance                          |
| loading  | `boolean` | `false`     |              | Show a loader instead of `children`             |
| disabled | `boolean` | `false`     |              | Change appearance and doesn't trigger `onClick` |
