# Title

A title allows to emphasis a text.

import { Alert, Title } from 'react-breeze';

<Title>Title (default)</Title>
<Title as="h2">Title (h2)</Title>
<Title as="h3">Title (h3)</Title>
<Title as="h4">Title (h4)</Title>
<Title as="h5">Title (h5)</Title>
<Title as="h6">Title (h6)</Title>

**Properties**

Title takes the same properties as a _heading_ element, but can also be given:

| Name | Type                                                                    | Default | Description |
| ---- | ----------------------------------------------------------------------- | ------- | ----------- |
| as   | `undefined` \| `'h1'` \| `'h2'` \| `'h3'` \| `'h4'` \| `'h5'` \| `'h6'` | `h1`    | Set heading |
