# Spinner

A spinner tells the user an action is still being executed.

<Alert className="mb-2" color="blue">
  <Spinner color="blue" inverted />
</Alert>
<Spinner color="red" />

```jsx
import { Alert, Spinner } from 'react-breeze';

<Alert color="blue">
  <Spinner color="blue" inverted />
</Alert>
<Spinner color="red" />
```

**Properties**

Spinner takes the same properties as a `div`, as well as:

| Name     | Type      | Default     | Required     | Description                                |
| -------- | --------- | ----------- | ------------ | ------------------------------------------ |
| color    | `string`  | `undefined` | **required** | Define spinner color                       |
| inverted | `boolean` | `false`     |              | Use given color inverted variation instead |
