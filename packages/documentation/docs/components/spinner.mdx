# Spinner

A spinner tells the user an action is still being executed.

<Alert className="mb-2">
  <Spinner />
</Alert>

```jsx
import { Alert, Spinner } from 'react-breeze';

<Alert>
  <Spinner />
</Alert>;
```

**Properties**

Spinner takes the same properties as a `div`.
