Button

A Button suggest a user action.

Color

import { Button } from 'react-breeze';

<Button>Button (Primary)</Button>
<Button color="secondary">Button (Primary)</Button>
<Button color="success">Button (Primary)</Button>
<Button color="info">Button (Primary)</Button>
<Button color="warning">Button (Primary)</Button>
<Button color="danger">Button (Primary)</Button>

Outline

Buttons can have a different appearance if they're outlined.

import { Button } from 'react-breeze';

<Button outline>Button (Primary)</Button>
<Button outline color="secondary">Button (Primary)</Button>
<Button outline color="success">Button (Primary)</Button>
<Button outline color="info">Button (Primary)</Button>
<Button outline color="warning">Button (Primary)</Button>
<Button outline color="danger">Button (Primary)</Button>

Loading

Buttons can display a loader, useful when working with async actions.

import { Button } from 'react-breeze';

<Button loading>Button (Primary)</Button>
<Button loading color="secondary">Button (Primary)</Button>
<Button loading color="success">Button (Primary)</Button>
<Button loading color="info">Button (Primary)</Button>
<Button loading color="warning">Button (Primary)</Button>
<Button loading color="danger">Button (Primary)</Button>

Disabled

Buttons can have a disabled state, used to indicate they cannot be clicked (default or outline).

import { Button } from 'react-breeze';

<Button disabled>Button (Primary)</Button>
<Button disabled color="secondary">Button (Primary)</Button>
<Button disabled color="success">Button (Primary)</Button>
<Button disabled color="info">Button (Primary)</Button>
<Button disabled color="warning">Button (Primary)</Button>
<Button disabled color="danger">Button (Primary)</Button>

<Button outline disabled>Button (Primary)</Button>
<Button outline disabled color="secondary">Button (Primary)</Button>
<Button outline disabled color="success">Button (Primary)</Button>
<Button outline disabled color="info">Button (Primary)</Button>
<Button outline disabled color="warning">Button (Primary)</Button>
<Button outline disabled color="danger">Button (Primary)</Button>

Properties

Button takes the same properties as a button, but can also be given:

NameTypeDefaultDescription
colorundefined | 'secondary' | 'success' | 'info' | 'warning' | 'danger'undefinedSet color
outlinebooleanfalseAlternative appearance
loadingbooleanfalseShow a loader instead of children
disabledbooleanfalseChange appearance and doesn't trigger onClick
