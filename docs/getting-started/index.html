Getting Started

react-breeze is a component library working with 6 variants: primary, secondary, success, info, warning, danger.

It allows for quick prototyping, using a set of configurable colors with a given intention.

Installation

yarn

yarn add react-breeze

npm

npm add --save react-breeze

Configuration

Before using react-breeze, you need to setup Tailwind CSS.

Next, we need to configure colors and their variants, as well as allowing the variant active for background color:

tailwind.config.js

  purge: [],
  darkMode: false,
  theme: {
    extend: {
      colors: {
        light: {
          DEFAULT: '#fff',
          600: '#e6e6e6',
